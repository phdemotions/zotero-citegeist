OpenAlex is the citation database Citegeist reads from. It has been metered since July 2026: without a key you get a free daily allowance, which is enough for normal use but can run out during a large library scan (error CG-API42). A free key from openalex.org raises it. The key is stored on this computer only — never synced, never logged, and only ever sent to OpenAlex. If something isn't working, copy the diagnostic report and paste it into your GitHub issue. It contains your Citegeist build, your Zotero version, and the problems recorded this session. It carries no titles, DOIs, or other library content, no API key, and no personal details such as your username.
