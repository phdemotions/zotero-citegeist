What is happening now

The top area has three separate bands. Each new control adds visual weight, so the toolbar becomes harder to scan as discovery filters grow.

What should change

Treat the top as a command bar: mode first, search second, sort/filter third, with only the controls needed for the current workflow.

What to defer

Additional discovery filters need their own design pass; they should not be squeezed into this top bar.
